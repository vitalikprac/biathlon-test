# Biathlon Test App

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install all dependencies.

```bash
npm install 
```
Or use the package manager [yarn](https://yarnpkg.com/)
```bash
yarn install 
```
## Start app

```bash
npm start
```
```bash
yarn start
```

## Creating an optimized production build

```bash
npm build
```
```bash
yarn build
```
